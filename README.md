## Minecraft_1.16.3_ZY_World_Chronicals

Minecraft 1.16.3我的世界游戏文件夹存档，我自己留存的。我非常喜爱这个游戏，尽管好久没玩了，但它对我来说仍然是我最重要的游戏，所以我想把它留存在互联网上。

release中是空白的1.16.3游戏文件，有非常多的光影可以选择，加了VoxelMap、OptiFabric、ItemScroller等基本模组和java环境，加的模组不会改变原版的游戏逻辑，但都是非常便利的工具。解压后点击HMCL启动器即可游玩，如果电脑里没有java可以先双击压缩包中的jre-8u261-windows-x64安装java环境，然后就可以进入游戏了。
